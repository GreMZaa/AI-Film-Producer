# 🎬 Garage Hollywood: AI Film Studio

Гибридная система для создания ИИ-фильмов. Бот живет в облаке (Vercel), а тяжелый рендеринг (GPU) происходит на вашем мощном ПК.

## 🚀 Быстрый запуск (Для Владельца)

### 1. Настройка ПК (Локальный сервер)
1.  Убедитесь, что у вас установлены **Ollama** и **ComfyUI**.
2.  Запустите `GarageHollywood.exe`. 
    *   *Скрипт сам найдет и запустит Ollama и ComfyUI.*
    *   *Скрипт автоматически создаст Ngrok-туннель.*
3.  **Важно:** Скопируйте `Public URL` из консоли (например, `https://abcd-123.ngrok-free.app`).

### 2. Настройка Облака (Vercel)
1.  Зайдите в панель управления **Vercel**.
2.  В настройках проекта (Environment Variables) обновите переменную `LOCAL_SERVER_URL`.
    *   Вставьте туда адрес из предыдущего шага.
3.  Перезапустите (Redeploy) проект на Vercel.

---

## 🛠 Компоненты системы

*   **Telegram Bot**: Обрабатывает команды пользователей, платежи и сценарии. Деплоится на Vercel.
*   **Local API Server**: Принимает запросы от бота, управляет ComfyUI и Ollama для генерации видео.
*   **Economy System**: 
    *   **Trial**: Бесплатные видео с вотермарком.
    *   **Indie Premiere**: Платная подписка. Убирает вотермарк и добавляет титры "Directed by [User]".
*   **Maintenance**: Автоматическая очистка временных файлов (`outputs/temp`) каждые 7 дней.

## 📦 Сборка своего EXE
Если вы внесли изменения в код и хотите собрать новый EXE:
```bash
python build_exe.py
```
Файл появится в папке `dist/GarageHollywood.exe`.
