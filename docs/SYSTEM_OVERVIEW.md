# 🎬 Garage Hollywood - Обзор системы

Этот документ содержит описание текущей архитектуры, возможностей и инфраструктуры проекта по генерации AI-фильмов.

---

## 🏗 Архитектура проекта

Система работает по гибридной схеме "Облако + Локальный сервер", что позволяет использовать мощь вашего ПК для тяжелых нейросетей, сохраняя доступность бота 24/7.

1. **Telegram Bot API**: Принимает команды от пользователя.
2. **Vercel (Cloud)**: Принимает вебхуки от Telegram и перенаправляет их на ваш локальный ПК через туннель.
3. **ngrok (Tunnel)**: Обеспечивает стабильный адрес (`venial-syringeful-bryan.ngrok-free.dev`) для связи облака с вашим компьютером.
4. **Local Server (FastAPI)**: "Мозг" системы, запущенный на вашем ПК. Управляет логикой генерации и общается с нейросетями.
5. **Ollama Engine**: Локальный движок для запуска больших языковых моделей (LLM).

---

## 🛠 Технический стек и Инфраструктура

*   **GPU**: NVIDIA GeForce RTX 5050 (используется для ускорения вычислений).
*   **Хранение моделей**: Вынесено на диск **D:\ollama_models**, чтобы не занимать место на системном диске C.
*   **LLM Модель**: `llama3.1` (8B) — используется для написания сценариев, обработки брифов и планирования кадров.
*   **Автоматизация**: Файл `run_server.bat` полностью автоматизирует запуск всей цепочки (Ollama -> ngrok -> FastAPI).

---

## ✨ Что система умеет СЕЙЧАС (Текущие возможности)

1. **Генерация сценариев**:
    *   Принимает текстовый бриф от пользователя в Telegram.
    *   Анализирует запрос и создает структурированный сценарий (через `llama3.1`).
2. **Интеллектуальный мост**:
    *   Автоматическая обработка вебхуков.
    *   Устойчивость к перезагрузкам (благодаря статическому домену ngrok).
3. **Управление ресурсами**:
    *   Автоматическое высвобождение портов при запуске.
    *   Контроль жизненного цикла Ollama (перезапуск с нужными переменными окружения).

---

## 🚀 Как пользоваться (Быстрый старт)

Для запуска всей системы достаточно запустить один файл:
`C:\Users\Admin\Desktop\Video\run_server.bat`

**Что произойдет после запуска:**
1. Закроются все старые копии Ollama и ngrok.
2. Подключится диск **D** для моделей.
3. Поднимется защищенный туннель.
4. Бот станет доступен в Telegram.
