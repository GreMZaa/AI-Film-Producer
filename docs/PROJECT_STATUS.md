# 📽️ Состояние проекта: «Гаражный Голливуд» (AI Film Producer)

Этот документ содержит обзор текущих возможностей системы, архитектуры и инструкцию по запуску.

## 🚀 Что уже работает

### 1. 🧠 Мозг (LLM)
*   **Модель:** Llama 3.1 (через Ollama).
*   **Персонаж:** Саркастичный инди-режиссер. Он не просто пишет сценарии, он критикует твой вкус и считает современные блокбастеры мусором.
*   **Функционал:** Превращает короткий бриф в полноценный сценарий (JSON) со сценами и техническими промптами.

### 2. 🎨 Визуальное производство (Image Gen)
*   **Движок:** ComfyUI API.
*   **Модель:** FLUX.1 [schnell] GGUF.
*   **Интеграция:** Автоматическая отправка промптов из сценария в ComfyUI.
*   **Система правок:** Кнопка `[🔄 Перегенерировать]` под каждым кадром в Telegram.

### 3. 🎬 Рендер-пайплайн (Alpha)
*   **Видео:** Анимация статичных кадров (SVD).
*   **Звук:** Озвучка диалогов (Bark).
*   **Сборка:** Автоматический монтаж в MP4 (MoviePy).

---

## 📂 Структура проекта
*   `src/bot/`: Код Telegram бота.
*   `src/local_server/`: Основной бэкенд (FastAPI).
*   `outputs/`: Результаты (images, videos, audio).

---

## 🛠️ Инструкция по запуску

1. **Запусти Ollama:** `ollama run llama3.1`
2. **Запусти ComfyUI:** (Порт 8188).
3. **Запусти Сервер:** `python -m src.local_server.main`
4. **Запусти Бота:** `python -m src.bot.main`
