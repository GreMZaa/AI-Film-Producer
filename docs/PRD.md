🎬 PRD: Telegram-бот «ИИ-Продюсер» (Local OS Edition)
1. Описание продукта (Concept)

«ИИ-Продюсер» — это Telegram-бот, представляющий собой автоматизированный пайплайн для создания коротких анимационных фильмов (преимущественно в стиле пластилиновой анимации). В отличие от классических агрегаторов (как SYNTX.AI), которые просто предоставляют доступ к инструментам, этот бот берет на себя роль режиссера, объединяя разрозненные нейросети в единую производственную цепочку.

2. Функционал и логика работы (Functionality)
Бот работает в пошаговом «Режиссерском режиме»:


Этап 1: Сценарий. На основе идеи пользователя локальная модель (Llama 3.1) генерирует диалоги и промпты для сцен. Доступно до 3 правок текста.


Этап 2: Раскадровка. Генерация статических кадров через Flux.1 [schnell]. Пользователь может точечно редактировать детали через встроенное Web App (Inpainting) или заказать платную перегенерацию кадра.


Этап 3: Рендер. Финальная сборка видео (SVD/AnimateDiff), озвучка (Bark) и липсинк происходят на локальном «железе» владельца бота.


Этап 4: Выдача. Отправка готового ролика пользователю.

3. Цель проекта (Goal)
Решить «боль» создателей контента, которым нужно получить полноценное видео с озвучкой и сюжетом без необходимости самостоятельно осваивать десятки разных интерфейсов и платить за множество подписок.

4. Фишки и конкурентные преимущества (Killer Features)

Автономность: Использование Open-Source моделей на собственном ноутбуке избавляет от корпоративной цензуры и зависимости от платных API сторонних сервисов.


Режиссерский UX: Пошаговое утверждение каждого этапа (текст -> кадры -> видео), что гарантирует предсказуемый результат.


Простая экономика: Отказ от запутанных «токенов» в пользу фиксированной оплаты за готовый проект (Pay-per-Project).

5. Дизайн и Tone of Voice (ToV)

Личность: Саркастичный инди-киношник в стиле Тарантино. Бот общается дерзко, использует киношный сленг (препродакшен, рендер, мотор) и может подкалывать пользователя за его идеи.


Интерфейс: Минималистичный чат с кнопками управления этапами и встроенное Mini App для рисования масок при редактировании кадров.

6. Для кого и почему создаётся?

Для кого: Блогеры, мемоделы, арбитражники и обычные пользователи, которым нужен быстрый и качественный видеоконтент.


Почему: Чтобы дать возможность создавать «искусство на коленке» с минимальным порогом входа ($2.99 за клип) и понятным результатом.

7. Технический стек и API

Модели: Llama 3.1 (текст), Flux.1 (визуал), SVD (видео), Bark (аудио), Wav2Lip (липсинк).


Связь: Обмен данными между облачным ботом и локальным ноутбуком через JSON API-запросы.
